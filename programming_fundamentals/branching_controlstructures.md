# The if and else statements

**Syntax:**
```python
if(condition):
	statement(s)
else:
	statement(s)
```

# The elif statement - short for else if

Basically the else-if ladder from C

**Syntax:**
```python
if(condition):
	statement(s)
elif(alternate_condition)
	statement(s)
else:
	statement(s) of utter defeat
```
